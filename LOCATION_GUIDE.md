# Руководство по определению локации для погоды

## Назначение

Функция определения локации нужна, чтобы приложение получало погоду для выбранного пользователем населенного пункта.

## Способы выбора локации

### 1. Автоматическое определение

1. Откройте главный экран или экран выбора локации.
2. Нажмите кнопку определения текущего местоположения.
3. Разрешите доступ к геолокации, если приложение запросит разрешение.
4. Приложение определит координаты устройства и выберет ближайший населенный пункт.
5. Выбранный город сохранится и будет использоваться для загрузки погоды.

### 2. Выбор на карте

1. Перейдите на экран выбора локации.
2. Откройте режим карты.
3. Нажмите на нужную точку на карте.
4. Приложение определит ближайший населенный пункт к выбранной точке.
5. Найденный город будет сохранен как текущая локация.

### 3. Поиск города вручную

1. Перейдите на экран выбора локации.
2. Введите название города в поле поиска.
3. Выберите подходящий город из списка.
4. Город сохранится и будет использоваться для прогноза погоды.

## Как используется выбранная локация

После выбора город сохраняется в памяти приложения. При загрузке погоды приложение использует координаты сохраненного города. Если город еще не выбран, по умолчанию используется Москва.

## Возможные проблемы

- Если приложение не может определить местоположение, проверьте разрешение на геолокацию.
- Если GPS или сетевое определение местоположения отключены, включите службы геолокации в настройках устройства.
- Если выбранная точка находится далеко от населенных пунктов, приложение может выбрать ближайший доступный город.
